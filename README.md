# Job Hunt Tracker

A sleek, modest, and futuristic frontend-only application to help users track and manage their job hunting journey efficiently. Built with usability and simplicity in mind, the app features both dark and light themes, intuitive controls, and AI-assisted tools to streamline the job application process.

## ✨ Features

- 🔄 **Theme Toggle** – Switch seamlessly between dark and light modes.
- 🗂 **Job Application Tracker** – Track applications by status: Applied, Interviewing, Offer, Rejected.
- 📝 **Detailed Application Logging** – Save company name, job title, resume version used, notes, and key dates.
- 📌 **Reminders & Follow-ups** – Set follow-up reminders to stay proactive.
- 📊 **Statistics Dashboard** – Visualize your progress with charts and trends.
- 🧠 **OpenAI Integration** – Get AI-powered resume tailoring suggestions and follow-up email drafts.
- 🏷 **Tags & Filters** – Organize entries with custom tags, sort, and filter.
- 🗂 **Kanban View & Timeline** – View your journey visually using boards or chronological timelines.
- 💾 **Local Storage Only** – All data is stored in-browser, no backend required.
